# ramador09.github.io

Personal landing page for Raymond Amador — served at
**https://ramador09.github.io/**.

A single hand-built static page (no framework, no build step): `index.html` +
`style.css`. It shares the visual identity (Fraunces / Source Serif 4 /
JetBrains Mono, ink `#16213e` + amber `#c0851a` on parchment) of the two course
companions it links to:

- Elementary Computational Physics
- Molecular and Materials Modelling

## Editing

- **Bio / text:** edit `index.html`.
- **Styling / colours:** edit `style.css` (design tokens are at the top).
- **CV:** replace `cv.pdf` (export from Overleaf) and commit.
- **Links:** LinkedIn / Scholar / GitHub are in the footer of `index.html`; the
  email is assembled in a small script at the bottom so the address stays out of
  the page source.

Push to `main` and GitHub Pages serves the change within a minute.
